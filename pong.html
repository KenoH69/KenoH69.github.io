How it works: If you start the game it will immediatelyd start. You can play it with 2 players on one laptop. The first player can use W and S and the second person can use the arrow up and arrow down. Both of the players have one paddel on each side, the player can use this to move it up and down. The ball will spawn in the middel and will go to the left or the right. The main goal is to score by the enemy. If you manage to score, the scoreboard will change if the left player scored, the left score will go up. And the first player that will get 9 points wins. And the game will start over. Pong Made with: I made this game in unity. I used the code C#. And I used the program Visual Studio 2019.
